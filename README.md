# José William - Landing Page de Atleta

## Sobre o Projeto

Landing page desenvolvida para o atleta de basquete José William, 18 anos. O objetivo do site é servir como portfólio pessoal e ferramenta de divulgação profissional, apresentando suas habilidades, conquistas e melhores momentos em quadra.

O projeto foi desenvolvido com foco em performance, design moderno e experiência do usuário, utilizando animações suaves e uma navegação fluida para destacar a trajetória do atleta.

## Demonstração

- Deploy: [https://josewilliam.vercel.app/](https://josewilliam.vercel.app/)
- Repositório: [https://github.com/teuaraujoo/atletaBahia](https://github.com/teuaraujoo/atletaBahia)

## Tecnologias Utilizadas

### Core
- HTML5
- CSS3
- JavaScript

### Bibliotecas e Frameworks
- **GSAP (GreenSock Animation Platform)** - Animações profissionais com ScrollTrigger
- **Lenis** - Smooth scroll suave e performático
- **Bootstrap** - Sistema de modal responsivo

### Ferramentas
- **Figma** - Design e prototipação
- **Vercel** - Deploy e hospedagem
- **PageSpeed Insights** - Análise e otimização de performance

## Funcionalidades

### Seções do Site

**Hero Section**
Apresentação impactante com informações principais do atleta e call-to-action.

**Sobre**
História e trajetória de José William no basquete, apresentando sua jornada e objetivos.

**Highlights**
Seção dedicada ao vídeo com as melhores jogadas do atleta, exibido através de modal do Bootstrap para melhor experiência visual.

**Galeria**
Coleção com mais de 8 fotos do atleta em ação durante jogos, mostrando diferentes momentos da sua carreira.

**Contato**
Informações para contato profissional e redes sociais.

### Recursos Técnicos

**Animações GSAP**
Implementação de scroll animations utilizando ScrollTrigger. Os componentes do site aparecem suavemente conforme o usuário rola a página, criando uma experiência imersiva e profissional.

**Smooth Scroll com Lenis**
Navegação fluida e suave entre as seções, proporcionando uma experiência premium de navegação.

**Modal Responsivo**
Sistema de modal do Bootstrap para exibição do vídeo de highlights, garantindo boa visualização em todos os dispositivos.

## Aprendizados e Desafios

### Performance e Otimização

Este projeto foi uma grande oportunidade de aprendizado sobre otimização web e métricas de performance. Após o deploy inicial, utilizei o PageSpeed Insights para análise detalhada e implementei melhorias significativas:

**Otimizações Implementadas:**
- Entrega otimizada de imagens
- Carregamento eficiente de CSS e JavaScript
- Redução de renderizações de bloqueio
- Otimização da exibição de fontes
- Minimização de reflow forçado
- Análise e otimização da árvore de dependência da rede

Essas otimizações melhoraram significativamente os tempos de carregamento e a experiência geral do usuário, conceitos que levarei para todos os projetos futuros.

- Análise de performance com ferramentas reais
- Otimização de recursos para web
- Compreensão de métricas de carregamento
- Workflow completo: design, desenvolvimento e otimização

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/teuaraujoo/atletaBahia.git
```

2. Entre na pasta do projeto:
```bash
cd atletaBahia
```

3. Abra o arquivo index.html no navegador ou utilize Live Server no VS Code.
